# RemHelper
一款Markdown笔记应用，支持使用艾宾浩斯遗忘曲线帮助复习与管理

## run
1. `git clone https://www.github.com/Pu-gayhub/RemHelper  --depth=1`
2. `cd RemHelper`
3. `python script/gen_uic.py`
4. `python src/main.py`

## build
1. `git clone https://www.github.com/Pu-gayhub/RemHelper --depth=1`
2. `cd RemHelper`
3. `python script/make.py`
4. run `dist\main.exe`
